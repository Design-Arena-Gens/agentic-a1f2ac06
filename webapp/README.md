## Agentic Assistant Showcase

Приложение визуально отвечает на вопрос «что ты умеешь?», демонстрируя ключевые сценарии работы агентного ассистента:

- инженерия и архитектура
- продуктовый контент и UX-копирайтинг
- аналитика и эксперименты

## Команды

```bash
# запуск dev-сервера
npm run dev

# проверка линтеров
npm run lint

# продакшен сборка и запуск
npm run build
npm run start
```

## Файлы

- `src/app/page.tsx` — интерактивные секции с фильтрацией компетенций.
- `src/app/globals.css` — глобальные стили с градиентами и цветовыми переменными.
- `src/app/layout.tsx` — метаданные и базовое оформление.

## Развертывание

Проект готов к публикации на Vercel. После `npm run build` используйте `vercel deploy --prod` (см. инструкцию в корне репозитория).
